# Workout Tracker

A personal workout tracking web application built with React. Log workouts, track progress over time, upload progress photos, and view your training history on a calendar.

## Features

- **Dashboard** — overview of your last workout, latest progress photo, and lifetime stats
- **Log Workout** — record exercises with sets, reps, and weight, plus optional cardio entries
- **Progress** — track weight progression for individual exercises over time with personal best tracking
- **Photos** — upload and manage progress photos with date, tags, and notes. Side by side comparison view included
- **History** — browse all logged workouts with expandable detail view, edit and delete support
- **Calendar** — visualise training frequency by month, click any day to view or log a workout

## Tech Stack

- React (Create React App)
- React Router for navigation
- localStorage for data persistence
- Font Awesome for icons
- Fully responsive, mobile first design

## Getting Started

### Prerequisites

- Node.js 14 or higher
- npm

### Installation
```bash
git clone https://github.com/McKwakkle/workout-tracker.git
cd workout-tracker
npm install
npm start
```

The app will open at `http://localhost:3000`

## Data Storage

All data is stored locally in your browser using localStorage. No account or internet connection is required. Clearing your browser data will erase your workout history.
